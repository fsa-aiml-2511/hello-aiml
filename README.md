# Hello AI/ML!

Welcome to your first GitHub assignment for the FSA AI/ML Bootcamp!

## Your Task

Edit the `hello_world.py` file to introduce yourself to the class.

### Steps

1. Open `hello_world.py` in your code editor
2. Replace `"Your Name Here"` with your actual name
3. Replace `"One fun fact about me"` with something interesting about yourself
4. Save the file
5. Commit and push your changes to GitHub

### Example

Before:
```python
name = "Your Name Here"
fun_fact = "One fun fact about me"
```

After:
```python
name = "Abishek"
fun_fact = "I love building AI solutions for healthcare!"
```

## How to Submit

Once you've made your changes:

```bash
git add .
git commit -m "Added my introduction"
git push
```

Then check your repository on GitHub to make sure your changes appear!
